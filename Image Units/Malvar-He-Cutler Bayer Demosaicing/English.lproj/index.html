This filter is a fast way to debayer raw grayscale image data into RGB on the GPU, usually from a machine vision camera.

Debayering is done using the Malvar-He-Cutler method.

The filter is an port by Ole Kristensen of the optimised GPU method described in the paper
Efficient, high-quality Bayer demosaic filtering on GPUs
by Morgan McGuire, published in Journal of Graphics Tools 2009.
